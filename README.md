# xMes 生产管理系统

## 项目简介

xMes是一个基于Django后端和Vue前端的生产管理系统，包含生产管理、销售管理和用户管理模块。

## 技术栈

- 后端: Django 5.2
- 前端: Vue 3 + TypeScript + Vite
- 数据库: SQLite (开发环境)

## 功能模块

- 基础数据管理
- 生产管理
- 销售管理
- 用户管理

## 安装指南

### 后端依赖安装

```bash
cd backend
pip install -r requirements.txt
```

### 前端依赖安装

```bash
cd frontend
npm install
```

## 运行项目

### 启动后端

```bash
cd backend
python manage.py runserver
```

### 启动前端

```bash
cd frontend
npm run dev
```
